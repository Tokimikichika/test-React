# Product Store - SPA приложение для управления продуктами

## Описание

Single Page Application для управления каталогом продуктов с возможностью просмотра, создания, редактирования и удаления продуктов.

## Технологии

- **Next.js 14** - React фреймворк
- **TypeScript** - типизация
- **Zustand** - управление состоянием
- **React Hook Form** - работа с формами
- **Zod** - валидация данных
- **DummyJSON API** - источник данных о продуктах

## Функционал

### Основные возможности

1. **Список продуктов** (`/products`)
   - Отображение всех продуктов в виде карточек
   - Лайк/дизлайк продуктов с визуальной индикацией
   - Удаление продуктов
   - Фильтр "Все продукты" / "Избранное"
   - Урезанный текст для одинаковой высоты карточек
   - Переход на детальную страницу по клику на карточку

2. **Страница продукта** (`/products/:id`)
   - Подробная информация о продукте
   - Дополнительные изображения
   - Возможность добавить в избранное
   - Кнопка возврата на главную страницу
   - Кнопка редактирования (для пользовательских продуктов)

3. **Создание продукта** (`/create-product`)
   - Форма с валидацией полей
   - Обязательные поля: название, описание, цена, бренд, категория, изображение
   - Сохранение в общий store

### Бонусные возможности

- ✅ **Пагинация** - постраничный вывод продуктов (12 на страницу)
- ✅ **Редактирование** - возможность редактировать пользовательские продукты
- ✅ **Дополнительная фильтрация** - фильтр по категориям
- ✅ **Поиск** - поиск по названию, описанию и бренду без кнопки отправки

## Установка и запуск

1. Установите зависимости:
```bash
npm install
```

2. Запустите проект в режиме разработки:
```bash
npm run dev
```

3. Откройте в браузере:
```
http://localhost:3000
```

## Особенности реализации

- **Управление состоянием**: Все данные хранятся в Zustand store
- **Валидация форм**: Используется react-hook-form + zod для валидации
- **Адаптивный дизайн**: Grid layout с автоматической адаптацией
- **TypeScript**: Полная типизация всего кода
- **Производительность**: Мемоизация фильтрованных данных с useMemo

## API

Используется публичное API DummyJSON:
- `GET https://dummyjson.com/products?limit=100` - получение списка продуктов

## Деплой на GitHub Pages

Проект настроен для автоматического деплоя на GitHub Pages.

### Первоначальная настройка:

1. Загрузите код в репозиторий GitHub:
```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/Tokimikichika/test-React.git
git push -u origin main
```

2. Настройте GitHub Pages в репозитории:
   - Перейдите в Settings → Pages
   - В разделе "Build and deployment" выберите "GitHub Actions" как источник

3. После push в main ветку автоматически запустится деплой

### Доступ к приложению:

После успешного деплоя приложение будет доступно по адресу:
**https://tokimikichika.github.io/test-React/**

### Локальная разработка:

```bash
# Для локальной разработки (без basePath)
npm run dev

# Для сборки production версии
npm run build
```
